Zenless Zone Zero

We have a mission to create a simple and easy to use store that the fans can order the merchandise of their choice. We would like to add merchandise like plushies, clothes, toys, and stickers. The project is another means through which we can show how a clean and functioning site can be created using HTML, CSS and JAVA.

This site will be the main place for fans to get merchandise. Our vision is to ensure the site is fully responsive and user-friendly on both desktop and mobile devices. Our focus is on a clean layout with branding and a better user experience for everyone.
